Este repositório tem o objetivo de fornecer um notebook que faz replicação do artigo:
DEMIGUEL, V.; GARLAPPI, L.; UPPAL, R. Optimal versus naive diversification: How inefficient is the 1/N portfolio strategy? The Review of Financial Studies, v. 22, n. 5, 2009, p. 1915–1953. DOI: 10.1093/rfs/hhm075.

Passos para execução no Google Colab

1) Faça o upload do arquivo DGU_2009_Replication_FF_Research_Data_Factors.ipynb para o Google Colab.
2) Abra o arquivo.
3) Faça o upload dos demais arquivos desse repositório para a pasta content.
4) Defina no código qual base de dados pretente utilizar alterando o valor da variável PATH_PANEL ="/content/F-F_Research_Data_Factors.txt" ou PATH_PANEL ="/content/F-F_Research_Data_Factors_2025.txt"
5) Execute todo o código do notebook.
